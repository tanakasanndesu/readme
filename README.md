■サービス概要
仕事、家事、育児で疲労やストレスが溜まったとき、誰かに話を聞いてもらいたいと思いませんか？そんなときは、AIが理解力豊かにあなたの相談や愚痴に耳を傾け、励ましやポジティブなサポートを提供します。

■ このサービスへの思い・作りたい理由
私は自分の気持ちや悩み、愚痴を誰かに話したいと思っても、身近に相談にのってくれる相手がいないことがありました。そこで、24時間いつでも話を聞いてくれて励ましてくれるAIが、その気持ちや悩みをユーザー同士で共有できるようなサービスを作りたいと思いました。

■ ユーザー層について
身近にすぐ相談できる相手がいない人。
話を聞いてもらいたい人

■サービスの利用イメージ
ユーザーは掲示板に投稿し、AIに悩みや相談事を共有します。それに対してAIが理解力を活かした前向きな気持ちになれる返答をします。ユーザー同士がお互いの悩みや経験を共有し、みんなが共感できる場を提供します。投稿を通じて、他のユーザーの悩みにも共感できます。

■ ユーザーの獲得について
自分の家族に使ってもらう。
周りの友人に使ってもらい、広げてもらう。
times,Xにて情報共有。

■ サービスの差別化ポイント・推しポイント
AIに相談するだけでなく、他のユーザーともコミュニケーションが図れる掲示板機能があります。ユーザー同士の交流を促進し、共感できる仲間ができることがこのサービスの強みです。
